# Smart-To-Do-List
A simple command-line To Do List manager built with Python.
This program allows users to add tasks, assign time (Morning/Afternoon/Evening), view taskes and remove tasks safely using error handling.

## Features
- Add new task
- Assign time to tasks (Morning/Afternoon/Evening)
- View task in a formatted table
- Remove tasks by number
- Handles invalid input using try/except
- Clean command-line menu interface

# Concepts Used
- Function
- Dictionaries
- Loops
- Conditional statements
- User input
- Error handling (try/except)
- enumerate() for indexing
- String formatting (f-strings)

## How to Run
- Download or clone the repository
- Navigate to the project folder
- Run the program
